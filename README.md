# PaleoAstroSim_10480BCE

محاكاة فلكية لتراصف المواقع المقدسة في مصر القديمة مع الأجرام السماوية عام 10480 قبل الميلاد.

## المحتويات
- `main.py`: الكود الرئيسي للمحاكاة
- `config.txt`: إعدادات المواقع والأهداف الفلكية
- `requirements.txt`: الحزم المطلوبة للتشغيل

## التشغيل
```bash
pip install -r requirements.txt
python main.py
